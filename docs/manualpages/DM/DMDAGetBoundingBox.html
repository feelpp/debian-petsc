DMDAGetBoundingBox

Synopsis

#include "petscdmda.h"   
PetscErrorCode  DMDAGetBoundingBox(DM da,PetscReal gmin[],PetscReal gmax[])
Collective on DMDA

Input Parameter

da -the distributed array

Output Parameters

gmin - global minimum coordinates (length dim, optional)
gmax - global maximim coordinates (length dim, optional)
