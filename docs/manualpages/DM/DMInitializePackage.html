DMInitializePackage

 This function initializes everything in the DM package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to AOCreate() or DMDACreate() when using static libraries.

Synopsis

PetscErrorCode  DMInitializePackage(void)

Keywords

AO, initialize, package
