DMMeshCreateGlobalRealVector

PetscErrorCode DMMeshCreateGlobalRealVector(DM dm, SectionReal section, Vec *gvec)
Collective on DMMesh

Input Parameters

mesh - the mesh object
section - The SectionReal

Output Parameters

gvec -the global vector
