DMPlexGetConeSize

 Return the number of in-edges for this point in the Sieve DAG

Synopsis

#include "petscdmplex.h"   
PetscErrorCode DMPlexGetConeSize(DM dm, PetscInt p, PetscInt *size)
Not collective

Input Parameters

mesh - The DMPlex
p - The Sieve point, which must lie in the chart set with DMPlexSetChart()

Output Parameter

size -The cone size for point p

See Also

DMPlexCreate(), DMPlexSetConeSize(), DMPlexSetChart()
