DMRestoreWorkArray

 Restores a work array guaranteed to be at least the input size, restore with DMRestoreWorkArray()

Synopsis

#include "petscdm.h"     
PetscErrorCode DMRestoreWorkArray(DM dm,PetscInt count,PetscDataType dtype,void *mem)
Not Collective

Input Parameters

dm - the DM object
count - The minium size
dtype - data type (PETSC_REAL, PETSC_SCALAR, PETSC_INT)

Output Parameter

array -the work array
