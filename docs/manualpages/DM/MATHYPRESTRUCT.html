MATHYPRESTRUCT

 MATHYPRESTRUCT = "hyprestruct" - A matrix type to be used for parallel sparse matrices based on the hypre HYPRE_StructMatrix.

Notes: Unlike the more general support for blocks in hypre this allows only one block per process and requires the block be defined by a DMDA.

The matrix needs a DMDA associated with it by either a call to MatSetupDM() or if the matrix is obtained from DMCreateMatrix()

See Also

MatCreate(), PCPFMG, MatSetupDM(), DMCreateMatrix()
