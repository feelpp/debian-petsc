PetscDrawResizeWindow

 Allows one to resize a window from a program.

Synopsis

#include "petscdraw.h" 
PetscErrorCode  PetscDrawResizeWindow(PetscDraw draw,int w,int h)
Collective on PetscDraw

Input Parameter

draw - the window
w,h - the new width and height of the window
