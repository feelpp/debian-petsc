PetscInitializeDrawPackage

 This function initializes everything in the PetscDraw package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the call to PetscInitialize() when using static libraries.

Synopsis

#include "petscdraw.h" 
PetscErrorCode  PetscDrawInitializePackage(void)

Keywords

Petsc, initialize, package
