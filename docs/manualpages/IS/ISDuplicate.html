ISDuplicate

 Creates a duplicate copy of an index set.

Synopsis

#include "petscis.h" 
PetscErrorCode  ISDuplicate(IS is,IS *newIS)
Collective on IS

Input Parmeters

is -the index set

Output Parameters

isnew -the copy of the index set

Notes

ISDuplicate() does not copy the index set, but rather allocates storage for the new one. Use ISCopy() to copy an index set.

See Also

ISCreateGeneral(), ISCopy()
