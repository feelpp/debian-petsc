DMKSPCopy

Synopsis

PetscErrorCode DMKSPCopy(DMKSP kdm,DMKSP nkdm)
Not Collective

Input Argument

kdm - Original DMKSP
nkdm - DMKSP to receive the data, should have been created with DMKSPCreate()
