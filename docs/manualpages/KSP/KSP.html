KSP

 Abstract PETSc object that manages all Krylov methods. This is the object that manages the linear solves in PETSc (even those such as direct solvers that do no use Krylov accelerators).

Synopsis

typedef struct _p_KSP*     KSP;

Notes: When a direct solver is used but no Krylov solver is used the KSP object is still used by with a KSPType of KSPPREONLY (meaning application of the preconditioner is only used as the linear solver).
