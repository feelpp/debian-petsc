KSPGetMonitorContext

 Gets the monitoring context, as set by KSPMonitorSet() for the FIRST monitor only.

Synopsis

PetscErrorCode  KSPGetMonitorContext(KSP ksp,void **ctx)
Not Collective

Input Parameter

ksp -iterative context obtained from KSPCreate()

Output Parameter
