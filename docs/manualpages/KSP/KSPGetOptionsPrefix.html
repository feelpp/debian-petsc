KSPGetOptionsPrefix

 Gets the prefix used for searching for all KSP options in the database.

Synopsis

#include "petscksp.h" 
PetscErrorCode  KSPGetOptionsPrefix(KSP ksp,const char *prefix[])
Not Collective

Input Parameters

ksp -the Krylov context

Output Parameters

prefix -pointer to the prefix string used is returned

Notes: On the fortran side, the user should pass in a string 'prifix' of sufficient length to hold the prefix.

Keywords

KSP, set, options, prefix, database

See Also

KSPSetOptionsPrefix(), KSPAppendOptionsPrefix()
