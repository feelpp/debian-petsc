KSPLSQRMonitorDefault

 Print the residual norm at each iteration of the LSQR method and the norm of the residual of the normal equations A'*A x = A' b

Synopsis

PetscErrorCode  KSPLSQRMonitorDefault(KSP ksp,PetscInt n,PetscReal rnorm,void *dummy)
Collective on KSP

Input Parameters

ksp - iterative context
n - iteration number
rnorm - 2-norm (preconditioned) residual value (may be estimated).
dummy - unused monitor context

Keywords

KSP, default, monitor, residual

See Also

KSPMonitorSet(), KSPMonitorTrueResidualNorm(), KSPMonitorLGResidualNormCreate(), KSPMonitorDefault()
