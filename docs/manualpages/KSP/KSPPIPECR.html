KSPPIPECR

 Pipelined conjugate residual method There method has only a single non-blocking reduction per iteration, compared to 2 blocking for standard CR. The non-blocking reduction is overlapped by the matrix-vector product, but not preconditioner application.

See also KSPPIPECG, where the reduction is only overlapped with the matrix-vector product.

Level

beginner

Notes

MPI configuration may be necessary for reductions to make asynchronous progress, which is important for performance of pipelined methods. See the FAQ on the PETSc website for details.

Reference
