KSPRichardsonSetScale

 Set the damping factor; if this routine is not called, the factor defaults to 1.0.

Synopsis

PetscErrorCode  KSPRichardsonSetScale(KSP ksp,PetscReal scale)
Logically Collective on KSP

Input Parameters
