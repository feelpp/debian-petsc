KSP_GMRES_CGS_REFINE_NEVER

 Do two steps of the classical (unmodified) Gram-Schmidt process.

Note: This is roughly twice the cost of KSP_GMRES_CGS_REFINE_NEVER because it performs the process twice but it saves the extra norm calculation needed by KSP_GMRES_CGS_REFINE_IFNEEDED.

You should only use this if you absolutely know that the iterative refinement is needed.
