MATNEST

 MATNEST = "nest" - Matrix type consisting of nested submatrices, each stored separately.

Notes

This matrix type permits scalable use of PCFieldSplit and avoids the large memory costs of extracting submatrices. It allows the use of symmetric and block formats for parts of multi-physics simulations. It is usually used with DMComposite and DMGetMatrix()
