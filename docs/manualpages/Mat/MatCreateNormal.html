MatCreateNormal

 Creates a new matrix object that behaves like A'*A.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatCreateNormal(Mat A,Mat *N)
Collective on Mat

Input Parameter

A -the (possibly rectangular) matrix

Output Parameter

N -the matrix that represents A'*A

Notes: The product A'*A is NOT actually formed! Rather the new matrix object performs the matrix-vector product by first multiplying by A and then A'
