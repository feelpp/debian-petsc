MatDenseRestoreArrayF90

 Restores a matrix array that has been accessed with MatGetArrayF90().

Synopsis

MatDenseRestoreArrayF90(Mat x,{Scalar, pointer :: xx_v(:)},integer ierr)
Not collective

Input Parameters

x - matrix
xx_v - the Fortran90 pointer to the array

Output Parameter

ierr -error code

Example of Usage

       PetscScalar, pointer xx_v(:)
       ....
       call MatDenseGetArrayF90(x,xx_v,ierr)
       a = xx_v(3)
       call MatDenseRestoreArrayF90(x,xx_v,ierr)

See Also
