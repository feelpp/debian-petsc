MatFindZeroDiagonals

 Finds all the rows of a matrix that have zero or no entry in the matrix

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatFindZeroDiagonals(Mat mat,IS *is)
Collective on Mat

Input Parameter

mat -the matrix

Output Parameter

is -if any rows have zero diagonals this contains the list of them
