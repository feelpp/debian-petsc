MatISSetLocalMat

 Set the local matrix stored inside a MATIS matrix.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatISSetLocalMat(Mat mat,Mat local)

Input Parameter

mat -the matrix
local -the local matrix usually MATSEQAIJ

Output Parameter

Notes

This can be called if you have precomputed the local matrix and want to provide it to the matrix object MATIS.
