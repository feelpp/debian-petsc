MatMFFDInitializePackage

 This function initializes everything in the MatMFFD package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to MatCreate_MFFD() when using static libraries.

Synopsis

#include "petscmat.h"   
PetscErrorCode  MatMFFDInitializePackage(void)

Keywords

Vec, initialize, package
