MatMultTransposeAdd

 Computes v3 = v2 + A' * v1.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatMultTransposeAdd(Mat mat,Vec v1,Vec v2,Vec v3)
Neighbor-wise Collective on Mat and Vec

Input Parameters

mat - the matrix
v1, v2 - the vectors

Output Parameters

v3 -the result

Notes

The vectors v1 and v3 cannot be the same. I.e., one cannot call MatMultTransposeAdd(A,v1,v2,v1).

See Also
