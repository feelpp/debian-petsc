MatSetLocalToGlobalMappingBlock

 Sets a local-to-global numbering for use by the routine MatSetValuesBlockedLocal() to allow users to insert matrix entries using a local (per-processor) numbering.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatSetLocalToGlobalMappingBlock(Mat x,ISLocalToGlobalMapping rmapping,ISLocalToGlobalMapping cmapping)
Not Collective

Input Parameters

x - the matrix
rmapping - row mapping created with ISLocalToGlobalMappingCreate() or ISLocalToGlobalMappingCreateIS()
cmapping - column mapping
