PCMGSetRhs

 Sets the vector space to be used to store the right-hand side on a particular level.

Synopsis

#include "petscksp.h" 
PetscErrorCode  PCMGSetRhs(PC pc,PetscInt l,Vec c)
Logically Collective on PC and Vec

Input Parameters

pc - the multigrid context
l - the level (0 is coarsest) this is to be used for
c - the space

Notes: If this is not provided PETSc will automatically generate one.

You do not need to keep a reference to this vector if you do not need it PCDestroy() will properly free it.

Keywords

MG, multigrid, set, right-hand-side, rhs, level

See Also
