PFApplyVec

 Applies the mathematical function to a vector

Synopsis

#include "petscpf.h" 
PetscErrorCode  PFApplyVec(PF pf,Vec x,Vec y)
Collective on PF

Input Parameters

pf - the function context
x - input vector (or NULL for the vector (0,1, .... N-1)

Output Parameter

y -output vector
