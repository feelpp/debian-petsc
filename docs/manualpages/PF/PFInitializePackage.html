PFInitializePackage

 This function initializes everything in the PF package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to PFCreate() when using static libraries.

Synopsis

#include "petscpf.h" 
PetscErrorCode  PFInitializePackage(void)

Keywords

Vec, initialize, package
