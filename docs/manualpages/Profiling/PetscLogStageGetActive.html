PetscLogStageGetActive

 Returns stage activity for PetscLogEventBegin() and PetscLogEventEnd().

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscLogStageGetActive(PetscLogStage stage, PetscBool  *isActive)
Not Collective

Input Parameter

stage -The stage

Output Parameter

isActive -The activity flag, PETSC_TRUE for logging, else PETSC_FALSE (defaults to PETSC_TRUE)

See Also

PetscLogStagePush(), PetscLogStagePop(), PetscLogEventBegin(), PetscLogEventEnd(), PetscPreLoadBegin(), PetscPreLoadEnd(), PetscPreLoadStage()
