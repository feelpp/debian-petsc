DMSNESGetFunction

 get SNES residual evaluation function

Synopsis

#include "petscsnes.h" 
#include "petscdm.h" 
PetscErrorCode DMSNESGetFunction(DM dm,PetscErrorCode (**SNESFunction)(SNES,Vec,Vec,void*),void **ctx)
Not Collective

Input Argument

dm -DM to be used with SNES

Output Arguments

SNESFunction - residual evaluation function
ctx - context for residual evaluation

Note

SNESGetFunction() is normally used, but it calls this function internally because the user context is actually associated with the DM.

See Also

DMSNESSetContext(), DMSNESSetFunction(), SNESSetFunction(), SNESFunction
