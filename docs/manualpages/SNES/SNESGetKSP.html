SNESGetKSP

 Returns the KSP context for a SNES solver.

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESGetKSP(SNES snes,KSP *ksp)
Not Collective, but if SNES object is parallel, then KSP object is parallel

Input Parameter

snes -the SNES context

Output Parameter

ksp -the KSP context

Notes

The user can then directly manipulate the KSP context to set various options, etc. Likewise, the user can then extract and manipulate the PC contexts as well.

Keywords

SNES, nonlinear, get, KSP, context

See Also

KSPGetPC(), SNESCreate(), KSPCreate(), SNESSetKSP()
