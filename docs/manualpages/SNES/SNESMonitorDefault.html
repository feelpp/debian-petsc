SNESMonitorDefault

 Monitors progress of the SNES solvers (default).

Synopsis

#include "petsc-private/snesimpl.h"   
PetscErrorCode  SNESMonitorDefault(SNES snes,PetscInt its,PetscReal fgnorm,void *dummy)
Collective on SNES

Input Parameters

snes - the SNES context
its - iteration number
fgnorm - 2-norm of residual
dummy - unused context

Notes

This routine prints the residual norm at each iteration.

Keywords

SNES, nonlinear, default, monitor, norm

See Also

SNESMonitorSet(), SNESMonitorSolution()
