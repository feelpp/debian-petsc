SNESSetFunctionNorm

 Sets the 2-norm of the current function computed using VecNorm().

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESSetFunctionNorm(SNES snes,PetscReal fnorm)
Collective on SNES

Input Parameter

snes -SNES context
fnorm -2-norm of function

Keywords

SNES, nonlinear, set, function, norm
