SNESSetLineSearch

 Sets the linesearch on the SNES instance.

Synopsis

#include "petscsnes.h"  
PetscErrorCode SNESSetLineSearch(SNES snes, SNESLineSearch linesearch)
Collective on SNES

Input Parameters

snes - iterative context obtained from SNESCreate()
linesearch - the linesearch object

Notes

Use SNESGetLineSearch() to retrieve the preconditioner context (for example, to configure it using the API).

Keywords

SNES, set, linesearch

See Also

SNESGetLineSearch()
