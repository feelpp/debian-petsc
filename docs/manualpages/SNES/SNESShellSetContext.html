SNESShellSetContext

 sets the context for a shell SNES

Synopsis

#include "petscsnes.h"   
PetscErrorCode  SNESShellSetContext(SNES snes,void *ctx)
Logically Collective on SNES

Input Parameters

snes - the shell SNES
ctx - the context

Fortran Notes: The context can only be an integer or a PetscObject unfortunately it cannot be a Fortran array or derived type.

See Also

SNESCreateShell(), SNESShellGetContext()
