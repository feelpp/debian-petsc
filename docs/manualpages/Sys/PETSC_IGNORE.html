PETSC_IGNORE

 same as NULL, means PETSc will ignore this argument

Note: accepted by many PETSc functions to not set a parameter and instead use some default

Fortran Notes: This macro does not exist in Fortran; you must use PETSC_NULL_INTEGER, PETSC_NULL_DOUBLE_PRECISION etc
