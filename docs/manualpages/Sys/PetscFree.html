PetscFree

 Frees memory

Synopsis

#include "petscsys.h"
PetscErrorCode PetscFree(void *memory)
Not Collective

Input Parameter

memory -memory to free (the pointer is ALWAYS set to 0 upon sucess)

Notes: Memory must have been obtained with PetscNew() or PetscMalloc()

See Also

PetscNew(), PetscMalloc(), PetscFreeVoid()
