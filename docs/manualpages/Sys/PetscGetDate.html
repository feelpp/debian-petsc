PetscGetDate

 Gets the current date.

Synopsis

PetscErrorCode  PetscGetDate(char date[],size_t len)
Not collective

Input Parameter

len -length of string to hold date

Output Parameter

date -the date

This function DOES make a system call and thus SHOULD NOT be called from an error handler.
