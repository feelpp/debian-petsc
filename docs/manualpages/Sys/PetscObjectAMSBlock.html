PetscObjectAMSBlock

 Blocks the object if PetscObjectAMSSetBlock() has been called

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscObjectAMSBlock(PetscObject obj)
Collective on PetscObject

Input Parameters

obj -the Petsc variable Thus must be cast with a (PetscObject), for example, PetscObjectSetName((PetscObject)mat,name);

See Also

PetscObjectSetName(), PetscObjectAMSViewOff(), PetscObjectAMSSetBlock()
