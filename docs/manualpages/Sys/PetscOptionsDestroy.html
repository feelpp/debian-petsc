PetscOptionsDestroy

 Destroys the option database.

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscOptionsDestroy(void)

Note

Since PetscOptionsDestroy() is called by PetscFinalize(), the user typically does not need to call this routine.
