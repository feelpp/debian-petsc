PetscOptionsEnd

 Ends a set of queries on the options database that are related and should be displayed on the same window of a GUI that allows the user to set the options interactively. Collective on the MPI_Comm used in PetscOptionsBegin()

Synopsis

#include "petscoptions.h"
PetscErrorCode PetscOptionsEnd(void)

Notes: Needs to be preceded by a call to PetscOptionsBegin()
