PetscSortIntWithArray

 Sorts an array of integers in place in increasing order; changes a second array to match the sorted first array.

Synopsis

#include "petscsys.h"  
PetscErrorCode  PetscSortIntWithArray(PetscInt n,PetscInt i[],PetscInt Ii[])
Not Collective

Input Parameters

n - number of values
i - array of integers
I - second array of integers

See Also

PetscSortReal(), PetscSortIntPermutation(), PetscSortInt()
