PetscStrendswith

 Determines if a string ends with a certain string

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscStrendswith(const char a[],const char b[],PetscBool *flg)
Not Collective

Input Parameters

a - pointer to string
b - string to endwith

Output Parameter

flg -PETSC_TRUE or PETSC_FALSE

Notes: Not for use in Fortran
