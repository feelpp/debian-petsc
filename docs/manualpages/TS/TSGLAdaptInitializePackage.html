TSGLAdaptInitializePackage

 This function initializes everything in the TSGLAdapt package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to TSCreate_GL() when using static libraries.

Synopsis

#include "petscts.h" 
PetscErrorCode  TSGLAdaptInitializePackage(void)
