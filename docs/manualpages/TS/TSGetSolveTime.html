TSGetSolveTime

 Gets the time after a call to TSSolve()

Synopsis

#include "petscts.h"  
PetscErrorCode  TSGetSolveTime(TS ts,PetscReal *ftime)
Not Collective

Input Parameter

ts - the TS context

Output Parameter

ftime - the final time. This time should correspond to the final time set with TSSetDuration()

Notes

Can only be called after the call to TSSolve() is complete.

Keywords

TS, nonlinear, set, convergence, test

See Also

TSSetConvergenceTest(), TSConvergedReason
