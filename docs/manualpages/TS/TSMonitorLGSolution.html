TSMonitorLGSolution

 Monitors progress of the TS solvers by plotting each component of the solution vector in a time based line graph

Synopsis

#include "petscts.h"  
PetscErrorCode  TSMonitorLGSolution(TS ts,PetscInt step,PetscReal ptime,Vec u,void *dummy)
Collective on TS

Input Parameters

ts - the TS context
step - current time-step
ptime - current time
lg - a line graph object

Notes: each process in a parallel run displays its component solutions in a separate window

Keywords

TS, vector, monitor, view

See Also

TSMonitorSet(), TSMonitorDefault(), VecView()
