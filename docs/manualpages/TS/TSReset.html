TSReset

 Resets a TS context and removes any allocated Vecs and Mats.

Synopsis

#include "petscts.h"  
PetscErrorCode  TSReset(TS ts)
Collective on TS

Input Parameter

ts -the TS context obtained from TSCreate()

Keywords

TS, timestep, reset
