TSSSPInitializePackage

 This function initializes everything in the TSSSP package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to TSCreate_SSP() when using static libraries.

Synopsis

#include "petscts.h"   
PetscErrorCode TSSSPInitializePackage(void)
