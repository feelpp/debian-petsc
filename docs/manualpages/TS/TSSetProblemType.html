TSSetProblemType

 Sets the type of problem to be solved.

Synopsis

#include "petscts.h"  
PetscErrorCode  TSSetProblemType(TS ts, TSProblemType type)
Not collective

Input Parameters

ts - The TS
type - One of TS_LINEAR, TS_NONLINEAR where these types refer to problems of the forms 
         U_t - A U = 0      (linear)
         U_t - A(t) U = 0   (linear)
         F(t,U,U_t) = 0     (nonlinear)

Keywords

TS, problem type
