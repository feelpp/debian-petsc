VecDestroyVecs

 Frees a block of vectors obtained with VecDuplicateVecs().

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecDestroyVecs(PetscInt m,Vec *vv[])
Collective on Vec

Input Parameters

vv - pointer to pointer to array of vector pointers
m - the number of vectors previously obtained

Fortran Note

The Fortran interface is slightly different from that given below. See the Fortran chapter of the users manual
