VecGhostIsLocalForm

 Checks if a given vector is the local form of a global vector

Synopsis

#include "petscvec.h"   
PetscErrorCode VecGhostIsLocalForm(Vec g,Vec l,PetscBool *flg)
Not Collective

Input Parameter

g - the global vector
l - the local vector

Output Parameter

flg -PETSC_TRUE if local vector is local form
