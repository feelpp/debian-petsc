VecInitializePackage

 This function initializes everything in the Vec package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to VecCreate() when using static libraries.

Synopsis

PetscErrorCode  VecInitializePackage(void)

Keywords

Vec, initialize, package
