VecWAXPY

 Computes w = alpha x + y.

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecWAXPY(Vec w,PetscScalar alpha,Vec x,Vec y)
Logically Collective on Vec

Input Parameters

alpha - the scalar
x, y - the vectors

Output Parameter

w -the result

Notes: w cannot be either x or y, but x and y can be the same

See Also

VecAXPY(), VecAYPX(), VecAXPBY()
