PETSC_VIEWER_AMS_

 Creates an AMS memory snooper PetscViewer shared by all processors in a communicator.

Synopsis

PetscViewer PETSC_VIEWER_AMS_(MPI_Comm comm)
Collective on MPI_Comm

Input Parameters

comm -the MPI communicator to share the PetscViewer

Notes

Unlike almost all other PETSc routines, PETSC_VIEWER_AMS_() does not return an error code. The window PetscViewer is usually used in the form 
      XXXView(XXX object,PETSC_VIEWER_AMS_(comm));

See Also

PETSC_VIEWER_AMS_WORLD, PETSC_VIEWER_AMS_SELF
